# تعليمات التثبيت لبرنامج AI-Video-Generator

## المتطلبات الأساسية
- تأكد من أنك قد قمت بتثبيت [Python](https://www.python.org/downloads/) (الإصدار 3.6 أو أعلى).
- تأكد من أنك قد قمت بتثبيت [Git](https://git-scm.com/downloads).

## خطوات التثبيت

1. **استنساخ المستودع:**  
   افتح موجه الأوامر أو Terminal واستخدم الأمر التالي:
   ```bash
   git clone https://github.com/hemamedra1986/AI-Video-Generator.git
   ```
   ينتج الأمر أعلاه مجلدًا جديدًا يسمى `AI-Video-Generator` يحتوي على جميع ملفات المشروع.

2. **تغيير الدليل إلى مجلد المشروع:**  
   بعد استنساخ المستودع، انتقل إلى مجلد المشروع باستخدام الأمر:
   ```bash
   cd AI-Video-Generator
   ```

3. **تثبيت المتطلبات:**  
   بعد ذلك، قم بتثبيت المكتبات الضرورية باستخدام:
   ```bash
   pip install -r requirements.txt
   ```

4. **إعداد البيئة:**  
   عليك إعداد الإعدادات اللازمة عن طريق نسخ ملف الإعدادات:
   ```bash
   cp .env.example .env
   ```
   ثم فتح ملف `.env` وتحديث القيم حسب الحاجة.

5. **تشغيل المشروع:**  
   الآن يمكنك تشغيل المشروع باستخدام:
   ```bash
   python main.py
   ```

باتباع هذه التعليمات، يجب أن تكون قادرًا على تثبيت وتشغيل برنامج AI-Video-Generator بنجاح.
