# إرشادات المساهمة في المشروع

شكرًا لاهتمامك بالمساهمة في مشروع AI-Video-Generator! إليك بعض الإرشادات للمساهمة:

## كيفية البدء
1. **قم بعمل Fork للمستودع:** اضغط على زر "Fork" أعلى الصفحة لإنشاء نسختك الخاصة من المشروع.
2. **Clone المستودع:** استخدم الأمر التالي لاستنساخ المستودع إلى جهازك:
   ```bash
   git clone https://github.com/hemamedra1986/AI-Video-Generator.git
   ```
3. **إنشاء فرع جديد:** تأكد من إنشاء فرع جديد لعملك باستخدام الأمر:
   ```bash
   git checkout -b اسم_الفرع
   ```

## تقديم المساهمات
- بعد إجراء التغييرات، تأكد من إضافة كل الملفات التي قمت بتعديلها:
  ```bash
  git add .
  ```
- اكتب رسالة التزام توضح التغييرات:
  ```bash
  git commit -m "رسالة الالتزام"
  ```
- ادفع التغييرات إلى GitHub:
  ```bash
  git push origin اسم_الفرع
  ```

## إنشاء طلب سحب
- بعد دفع التغييرات، افتح واجهة GitHub الخاصة بمستودعك، وسترى خيارًا لإنشاء "Pull Request".
- حدد الفرع الذي عملت عليه وحدد الفرع الرئيسي للمشروع كمستودع الهدف.

## إرشادات واجب الالتزام
- تأكد من كتابة كود نظيف ومتعمد.
- أضف تعليقات توضيحية عند الحاجة.
- تأكد من أن كل الاختبارات تمر بنجاح قبل تقديم طلب السحب.

## أسئلة أو استفسارات
إذا كان لديك أي أسئلة، لا تتردد في فتح قضية جديدة في المستودع.

شكرًا لمساهمتك!
